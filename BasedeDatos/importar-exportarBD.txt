Para importar una base de datos debes de usar el comando de mysql como sigue:

Linux

mysql -u root -p -h localhost BaseLogin < NombredelaBD.sql

Para exportar una base de datos debes de usar el siguiente comando de mysql como sigue:

mysql -u root -p -h localhost BaseLogin > NombredelaBD.sql

Windows

Puedes usar phpmyAdmin y seguir los pasos de esta pagina:
